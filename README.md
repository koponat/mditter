# mditter
让你的Minecraft客户端可以不向服务器发送数据包响应, 你可以在这期间干任何事情 也可以是HACK的攻击

下载mditter

https://github.com/koponat/mditter/releases

你需要用到fabric mod加载器和fabric api进行搭载mditter
