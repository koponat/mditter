# MDITTER v1.5

MDITTER 是一款专为 Minecraft 开发的 Fabric 轻量化功能模组，专注于数据包分析与安全控制。

## 核心功能
* **欢迎界面**：全新设计的 v1.5 交互 UI，启动即见。
* **数据包拦截 (Packet Blocker)**：深度拦截客户端发往服务器的特定数据包，实现同步挂起。
* **抓包器 (Packet Sniffer)**：内置实时抓包工具（开发中），监控网络封包走向。
* **快捷访问**：直连 GitHub 仓库，随时获取最新更新与文档。
* **双语一键切换**：支持简体中文与英文的无缝切换。

## 运行环境
* **游戏版本**：1.21.1
* **加载器**：Fabric Loader (>=0.16.0)
* **依赖项**：Fabric API (>=0.102.0+1.21.1)
* **Java 版本**：Java 21

## 使用说明
1. **启动按键**：
   * **旧版 (v1.4)**：使用右侧 `Shift` 键（RSHIFT）打开。
   * **新版 (v1.5)**：统一改为键盘 **`J`** 键呼出主界面。
2. **操作逻辑**：在 UI 界面中通过点击各功能按钮即可执行拦截、切换语言或访问仓库。
